# Snake Game

A classic Snake Game built using Python and Pygame. Enjoy the retro gaming experience!

## Features

- Snake movement
- Food spawning
- Score tracking
- Collision detection
- Wall collision
- Self collision

## Technologies Used

- Python 3
- Pygame

## How to Run

Install pygame:

```bash
py -m pip install pygame
```

Run the game:

```bash
py main.py
```
